# English version
# FDD with FCCU Data in Process Automation

This repository contains introductory teaching materials for fault detection and diagnosis (FDD) with Fluid Catalytic Cracking Unit (FCCU) process data. The materials focus on multivariate process monitoring, anomaly detection, novelty detection, and the evaluation of data-driven FDD methods for continuous process systems.

The notebooks are designed as teaching and self-study material for applied AI, process automation, data science, and predictive maintenance. The focus is on understanding FDD workflows, preparing process data for monitoring tasks, and comparing classical and machine-learning-based methods.

## Target audience

The material is intended for students in process automation, applied AI, data science, and predictive maintenance.

Basic knowledge of Python, Jupyter Notebooks, statistics, machine learning, and process monitoring is helpful.

## Contents

The repository covers:

- fault detection and diagnosis with FCCU process data,
- normal operation and faulty operating conditions,
- exploratory analysis of multivariate process time series,
- data preprocessing and scaling,
- train/validation/test splitting for monitoring workflows,
- threshold calibration on normal operating condition data,
- false alarm rate and detection delay,
- dynamic PCA and multivariate process monitoring,
- autoencoder-based novelty detection,
- comparison of classical and machine-learning-based FDD methods,
- visualization and interpretation of monitoring scores and alarms.

# Deutsche Version
# FDD mit FCCU-Daten in der Prozessautomation

Dieses Repository enthält einführende Lehrmaterialien zur Fault Detection and Diagnosis (FDD) mit Prozessdaten einer Fluid Catalytic Cracking Unit (FCCU). Im Mittelpunkt stehen multivariate Prozessüberwachung, Anomalieerkennung, Novelty Detection und die Bewertung datengetriebener FDD-Methoden für kontinuierliche Prozesssysteme.

Die Notebooks sind als Lehr- und Selbstlernmaterial für angewandte KI, Prozessautomation, Data Science und Predictive Maintenance konzipiert. Der Fokus liegt auf dem Verständnis von FDD-Workflows, der Vorbereitung von Prozessdaten für Monitoring-Aufgaben und dem Vergleich klassischer sowie Machine-Learning-basierter Methoden.

## Zielgruppe

Das Material richtet sich an Studierende in Prozessautomation, angewandter KI, Data Science und Predictive Maintenance.

Grundkenntnisse in Python, Jupyter Notebooks, Statistik, Machine Learning und Prozessüberwachung sind hilfreich.

## Inhalte

Das Repository behandelt:

- Fault Detection and Diagnosis mit FCCU-Prozessdaten,
- Normalbetrieb und fehlerhafte Betriebszustände,
- explorative Analyse multivariater Prozesszeitreihen,
- Datenvorverarbeitung und Skalierung,
- Train-/Validation-/Test-Aufteilung für Monitoring-Workflows,
- Schwellwertkalibrierung auf Normaldaten,
- Fehlalarmrate und Erkennungsverzögerung,
- Dynamic PCA und multivariate Prozessüberwachung,
- Autoencoder-basierte Novelty Detection,
- Vergleich klassischer und Machine-Learning-basierter FDD-Methoden,
- Visualisierung und Interpretation von Monitoring-Scores und Alarmen.
